\begin{slide}
{\small
Conclusões
\begin{itemize} 
 \item É possível prever quando um escravo estará livre (tempo de processamento). 
Calcular os tempos de transmissão para que os escravos sempre estejam livres. 
\begin{displaymath}
n = \frac{tempo\ de\ processamento}{tempo\ de\ trasmiss\tilde{a}o\ para\ outros}
\end{displaymath}
 \item No caso da GDU $n = \frac{200}{27} = 7,41 \Rightarrow$ \eng{speed-up} máximo
 \item No caso do 2\eiro nível $ n = 2 $ satisfaz taxa máxima
\end{itemize}}

\begin{slide}
Conclusões finais
\begin{itemize} 
 \item A rede de decisões globais tem \eng{speed-up} máximo de 7,4
 \item É capaz de processar uma RoI em 27$\mu$s = 7,4KHz (evento)
 \item Isto não satisfaz o ambiente do segundo nível, porém é o máximo a ser 
atingido com esta tecnologia
 \item O \eng{speed-up} máximo para o 2\eiro nível é 3,4
 \item Isto não satisfaz o funcionamento para este tipo de processamento, mas também
 representa o máximo desta tecnologia
\end{itemize}
